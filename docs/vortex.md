## Setup

* python install package

    ```sh
    uv venv --python 3.13
    uv pip install vortex-data duckdb pyarrow
    ```

## Example

* create simple vortex

    ```sh
    source ~/.venv/bin/activate
    python -m src.example_vortex
    ```
